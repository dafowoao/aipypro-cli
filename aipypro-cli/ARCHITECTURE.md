# AiPyPro Pro v4 — 架构文档

## 模块

```
index.js         入口（ask / chat / history / tools / export / list）
lib/
├── config.js    配置加载
├── api.js       API 客户端（流式 chatStream + 非流式 chat）
├── agent.js     智能体循环
├── tools.js     15 个工具插件加载器
├── ui.js        终端界面（Markdown 渲染 / 状态栏 / 彩色主题）
└── history.js   会话历史 + 命名会话管理
```

## 对话界面（v4 特大升级）

### 多行输入
- 输入 `"""` 进入多行模式
- 再次输入 `"""` 或空行结束
- 支持粘贴多段代码

### 命令历史
- ↑/↓ 箭头键浏览命令历史
- Tab 键补全命令

### 新增聊天命令

| 命令 | 功能 |
|------|------|
| `/save <名称>` | 保存当前会话为命名会话 |
| `/load <名称>` | 加载命名会话 |
| `/list` | 列出所有命名会话 |
| `/session` | 查看当前会话详情 |
| `/tokens` | 查看 Token 用量 |
| `/model <模型>` | 切换模型 |
| `/export` | 导出会话为 Markdown |

### Markdown 渲染
- 标题 (# ## ###)
- 粗体 **text** / 斜体 *text*
- 行内代码 `code`
- 链接 [text](url)
- 无序/有序列表
- 引用 >
- 表格 | col | col |
- 分隔线 ---
- 20+ 语言代码块语法高亮

## 聊天命令

| 命令 | 功能 |
|------|------|
| `ask <问题>` | 直接提问（流式输出） |
| `chat` | 交互对话（自动保存历史） |
| `/help` | 帮助 |
| `/tools` | 列出工具 |
| `/history` | 查看历史 |
| `/stats` | 会话统计 |
| `/session` | 会话详情 |
| `/save` | 保存会话 |
| `/load` | 加载会话 |
| `/list` | 列出保存的会话 |
| `/export` | 导出为 Markdown |
| `/model` | 切换模型 |
| `undo` | 撤回 |
| `clear` | 清屏 |
| `exit` | 退出 |
